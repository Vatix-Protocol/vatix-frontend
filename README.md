# Vatix Frontend

Web interface for the Vatix prediction market protocol on Stellar.

## Overview

A modern, responsive web application for interacting with Vatix prediction markets.

## Features

- **Browse Markets**: Discover and filter prediction markets
- **Trade**: Buy and sell outcome shares
- **Create Markets**: Propose new prediction markets
- **Portfolio**: Track your positions and history
- **Wallet Integration**: Connect with Stellar wallets

## Tech Stack

- **Framework**: Next.js / React
- **Styling**: Tailwind CSS
- **Wallet**: Stellar Wallet SDK
- **State**: React Query
- **Charts**: Recharts / Chart.js
